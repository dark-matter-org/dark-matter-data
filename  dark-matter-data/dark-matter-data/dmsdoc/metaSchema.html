ActionTriggerInfo
Description The ActionTriggerInfo class simply provides a common base for the bundle of information required to remotely trigger an action on an object. Derivatives of this class are generated as part of the DMO generation process and have action specific interfaces to handle the parameters required by an action.

This mechanism is a replacement for the concept that ActionRequests were EXTENSIBLE objects. This way, the user can ask an object for the ActionTriggerInfo (ATI) for a particular action and have a well defined interface to specify the parameters for the action.
