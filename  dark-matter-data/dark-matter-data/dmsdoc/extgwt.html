MvcAction
Description The MvcAction class allows for the definition of behaviour that can be associated with menus, buttons and other UI components. Actions are associated with controllers so that they can be informed of events in which they're interested, since only controllers can register for events. Actions may be defined and used by menu items defined in other Controllers.
